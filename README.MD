# **How to install and use the app**

___
- ## Check if python3 is installed

```bash
python3 --version
```

- ## If it is not, run 

```bash
sudo apt update && sudo apt upgrade -y
sudo apt install python3
```

___

- ## Install the project libraries

```bash
pip install -e .
```

___ 

- ## Run the app
```bash
flask run
```
